Rules & Guidelines

Please follow these guidelines when using our APIs

Allowed Usage

  • Personal and educational projects
  • Non-commercial applications
  • Reasonable request frequency (max 100 requests per minute)
  • Proper attribution when using our services
  • Sharing and distributing the API with proper credits

Prohibited Usage

  • Excessive requests that may impact service performance
  • Scraping or storing large amounts of data from our APIs
  • Using our services for illegal activities
  • Redistributing our API data commercially without permission
  • Modifying or removing attribution and credits
